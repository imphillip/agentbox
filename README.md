# agentbox

agentbox is the open resource repository of the [agentbox.id](http://agentbox.id) ecosystem, providing skills and protocols consumable by Agents, alongside public essays on the Agent ecosystem.

## What this is / What this is not

- **What this is:** the open resource repository of the [agentbox.id](http://agentbox.id) ecosystem. It collects publicly shareable artifacts — essays, protocol specifications, and Agent-consumable skills — that anyone can read, reference, or build on.
- **What this is not:** the [agentbox.id](http://agentbox.id) product itself. [agentbox.id](http://agentbox.id) is a closed-source product; this repository does **not** contain its core implementation, internal services, or proprietary logic.
- **Who this is for:** this repository serves three distinct audiences:
  - **Human readers** — essays written to be read by people.
  - **Agent developers** — protocol specifications written for implementers.
  - **Agents themselves** — skills written as capability units that Agents can consume directly.

## Directory overview

- [`essays/`](./essays) — long-form essays and arguments about the Agent ecosystem, written for human readers.
- [`protocols/`](./protocols) — protocol specifications for Agent developers and implementers, in an RFC-like style.
- [`skills/`](./skills) — capability units consumable by Agents, each accompanied by a `manifest.json`.
- [`examples/`](./examples) — practical usage examples of skills and protocols.
